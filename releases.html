Releases

Vespa is released daily Mon-Thu, with some restrictions:

  • The release must pass all tests and run without regressions for select production applications
  • The release is not manually blocked / holiday block window

For each release of Vespa we provide the following artifacts:

  • Java artifacts for building Vespa applications on Maven Central
  • Vespa RPMs on Fedora Copr
  • Container images on Docker Hub

Vespa only releases from the current major version with minor version increments.

Vespa uses semantic versioning. It is normal to find "holes" in the version sequence, as releases are skipped.
